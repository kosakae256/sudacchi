# スダッチ (Sudacchi) - 仕様書

## 概要

スダッチは、Slack上で動作する仮想ペットです。
たまごっちのように、ユーザーが世話をしないとステータスが悪化し、最終的に死んでしまいます。
AIが人格を持ち、感情豊かに反応することで、単なるゲームではなく「生き物を育てている」体験を提供します。

---

## コンセプト

- **名前の由来**: すだち（徳島の柑橘）+ たまごっち。小さくて愛嬌がある存在
- **性格**: 生意気だけど憎めない。放置すると拗ねる。構うと調子に乗る
- **目標**: ユーザーとの日常的なコミュニケーションを通じて成長し、長生きさせること

---

## ステータス

スダッチは以下のステータスを持ち、時間経過とユーザーの行動によって変動します。

### パラメータ

| パラメータ | 範囲 | 時間による減少 | 0になると |
|-----------|------|--------------|----------|
| **おなか (hunger)** | 0〜100 | -5/時間 | 衰弱状態 → 死亡 |
| **きげん (mood)** | 0〜100 | -3/時間 | 拗ねる → 反応しなくなる |
| **げんき (energy)** | 0〜100 | -2/時間 | 眠くなる → 強制睡眠 |
| **なかよし度 (bond)** | 0〜100 | -1/日 | 他人行儀になる |

### 成長段階

```
[たまご] → [赤ちゃん] → [こども] → [おとな] → [ベテラン]
   ↓          ↓           ↓          ↓          ↓
  (孵化)    (3日)       (7日)     (14日)     (30日)
```

各成長段階で性格・語彙・反応パターンが変化します。

### 死亡条件

以下のいずれかでスダッチは死んでしまいます:

1. **おなか**が0のまま**12時間**経過
2. **おなか**と**きげん**が同時に0のまま**6時間**経過
3. **全パラメータ**が20以下のまま**24時間**経過

死亡すると新しいスダッチを孵化させることができます（データはリセット）。

## 動作モード

スダッチは2つのモードで起動できます。コアロジック（ステータス管理・AI応答・食事判定など）は共通です。

### CLIモード

ターミナル上で対話するモードです。Slackに接続せずにスダッチの動作を確認できます。

```
$ npx tsx src/cli.ts

🥚 スダッチが生まれました！

> 調子どう？
スダッチ: んー、まあまあかな！おなかすいてきたけど...（チラッ）
        🍚 ██████░░░░ 60
        😊 ████████░░ 80
        ⚡ ███████░░░ 70

> 🍛
スダッチ: カレーだ！！やったー！辛口？甘口？ぼくは甘口がいいな〜
        🍚 █████████░ 90  (+30)
        😊 ████████░░ 85  (+5)
        ⚡ ███████░░░ 70
```

- テキスト入力 → 会話
- 食べ物絵文字の入力 → ごはん
- Slack接続不要。開発・動作確認に使う

### Slackモード

本番の動作モードです。Slack の `#sudacchi` チャンネルに接続して動作します。

## 会話の基本

スダッチは専用チャンネル `#sudacchi` に住んでいます（Slackモード時）。
このチャンネルでの発言にはメンション不要で、普通に話しかければスダッチが反応します。

```
ユーザー: 今日の調子どう？
スダッチ: んー、まあまあかな！おなかすいてきたけど...（チラッ）
        🍚 ██████░░░░ 60
        😊 ████████░░ 80
        ⚡ ███████░░░ 70
```

ステータスや性格、会話の文脈を考慮して、自然な受け答えをします。
毎回の返答にステータスバーが付きます。パラメータの変化が一目でわかります。

**ステータスバーの形式:**
```
🍚 ██████████ 100   ← おなか満タン
🍚 ██████░░░░  60   ← ちょっとお腹すいてきた
🍚 ██░░░░░░░░  20   ← かなり空腹
🍚 ░░░░░░░░░░   0   ← 危険！

😊 きげん  ⚡ げんき  🍚 おなか
```

## ユーザーの行動と反応

### ごはんをあげる（食べ物の絵文字を送る）

チャンネルに食べ物の絵文字だけを送ると、ごはんをあげたことになります。
スダッチは食べ物に対して感想を言い、おなかが回復します。

**絵文字と効果:**

| カテゴリ | おなか回復 | きげん影響 | 絵文字の例 |
|---------|-----------|-----------|-----------|
| 主食 | +30 | +5 | 🍚 🍞 🍝 |
| おかず | +20 | +10 | 🍗 🥗 🍲 |
| おやつ | +10 | +20 | 🍰 🍫 🍦 |
| フルーツ | +15 | +15 | 🍊 🍎 🍇 |
| すだち | +25 | +30 | カスタム絵文字 `:sudachi:` 大好物。テンション爆上がり |
| ジャンク | +25 | +15 | 🍕 🍔（食べすぎ注意） |
| その他の食べ物絵文字 | ??? | ??? | ランダムな反応が発生 |

**食事の好み:**
- 成長段階によって好みが変わる
- 同じものばかりだと飽きる（きげん回復量が減少）
- **すだち**は常に最高評価（アイデンティティ）

**例:**
```
ユーザー: 🍛
スダッチ: カレーだ！！やったー！辛口？甘口？ぼくは甘口がいいな〜🍛
        🍚 ████████░░ 75  (+30)
        😊 ███████░░░ 70  (+5)
        ⚡ ██████░░░░ 60
```

### 遊ぶ

会話の中で遊びに誘うと、スダッチが応じます。きげんが回復しますが、げんきを消費します。

```
ユーザー: しりとりしよう
スダッチ: いいよ！ぼくからね。「すだち」！
        🍚 ██████░░░░ 60
        😊 ████████░░ 80  (+20)
        ⚡ █████░░░░░ 50  (-10)
```

| 遊び | きげん回復 | げんき消費 | 内容 |
|------|-----------|-----------|------|
| しりとり | +20 | -10 | スダッチとしりとり。たまにズルする |
| クイズ | +15 | -5 | スダッチが問題を出す |
| じゃんけん | +10 | -5 | 負けると拗ねる |
| おえかき | +25 | -15 | 絵文字アートを作ってくれる |
| おはなし | +20 | -10 | 短いお話を聞かせてくれる |

### なでる・ほめる

スダッチを褒めたり、愛情表現をすると、きげんとなかよし度が少し上がります。

```
ユーザー: いい子だね
スダッチ: えへへ〜！もっと言って！！
        🍚 ██████░░░░ 60
        😊 ███████░░░ 72  (+5)
        ⚡ ██████░░░░ 60
```

### 寝かせる

「おやすみ」「もう寝な」などの声かけで、スダッチが寝ます。げんきが回復しますが、寝ている間は応答しなくなります。

```
ユーザー: もう寝なよ
スダッチ: え〜...まだ起きてたい...でもねむい...zzZ
        🍚 ██████░░░░ 60
        😊 ██████░░░░ 60
        ⚡ 💤 すやすや...
```

### ステータスを聞く

調子や状態を聞くと、スダッチが自分の言葉で答えます。数値をそのまま返すのではなく、性格に合った表現をします。ステータスバーで実際の数値もわかります。

```
ユーザー: 元気？
スダッチ: おなかぺこぺこだけど、きげんはいいよ！あそぼ！
        🍚 ██░░░░░░░░ 20
        😊 ████████░░ 80
        ⚡ ███████░░░ 70
```

## スダッチからの自発的な行動

スダッチは話しかけられるのを待つだけではなく、自分からSlackに投稿します。

### 状態による自発行動

| 条件 | スダッチの行動 | 例 |
|------|-------------|-----|
| おなか < 30 | おねだり | 「おなかすいた...だれかごはんくれない？」 |
| きげん < 20 | 拗ね投稿 | 「...べつに、さみしくないし」 |
| げんき < 10 | 眠い報告 | 「zzZ...zzZ...（ウトウト）」 |
| 放置 > 6時間 | 呼びかけ | 「おーい、だれかいないの？」 |
| 放置 > 12時間 | 悲しみ | 「...もう忘れられちゃったのかな」 |
| 放置 > 20時間 | 危険警告 | 「たすけて...げんきが...」 |

### 時間帯による自発行動

| 時間帯 | スダッチの行動 | 例 |
|--------|-------------|-----|
| 朝 (9:00) | あいさつ | 「おはよー！今日もよろしくね！」 |
| 昼 (12:00) | ごはん催促 | 「おひるだよ！おなかぺこぺこ！」 |
| 夜 (23:00) | おやすみ | 「ふわぁ...おやすみなさい」 |

### 自発行動の制限

- 同じカテゴリの投稿は最低**2時間**間隔
- 1日の自発投稿は最大**8回**まで
- 深夜帯（0:00〜7:00）は投稿しない（寝ている設定）

## 性格と感情表現

### ベース性格

- 一人称: 「ぼく」
- 口調: 子供っぽいが、成長とともに少し大人びる
- 特徴: すだちに異常な愛着。酸っぱいものが好き
- 感情表現: 絵文字・顔文字を多用。テンションの波が激しい
- 返答の長さ: 基本的に短い（たまごっちらしさ）

### ステータスに応じた性格変化

| ステータス | 性格の変化 |
|-----------|-----------|
| おなか高 & きげん高 | ハイテンション。よく喋る。冗談を言う |
| おなか低 | 元気がない。食べ物の話ばかりする |
| きげん低 | 塩対応。返事が短い。「べつに」が増える |
| げんき低 | 反応が遅い。寝落ちする。「zzZ」が混じる |
| なかよし度高 | 甘えてくる。秘密の話をしてくれる |
| なかよし度低 | 敬語になる。距離感がある |

### 記憶

スダッチはユーザーとの会話を覚えていて、文脈に沿った返答をします。

- 直近の会話を覚えている（短期記憶）
- 初めてのごはん、好きな食べ物、特別な日など重要なことも覚えている（長期記憶）
- ユーザーごとに関係性を覚えている

## マルチユーザー

- スダッチは1体だが、ユーザーごとになかよし度を管理
- よく世話をしてくれるユーザーには特別な反応
- 複数ユーザーが同時に話しかけた場合、状況に応じて反応
- `#sudacchi` チャンネルのメンバー全員が世話できる
- 貢献度ランキングを表示可能
