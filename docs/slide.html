スダッチとは？

Slackチャンネルに住むAI仮想ペット。
たまごっちのように世話をしないとステータスが悪化し、最終的に死んでしまいます。

AIが人格を持ち、感情豊かに反応することで、
単なるゲームではなく「生き物を育てている」体験を提供します。

ステータスシステム

🍚 おなか (hunger)
時間経過: -5 / 時間
0のまま12時間で死亡
😊 きげん (mood)
時間経過: -3 / 時間
0になると拗ねて反応しなくなる
⚡ げんき (energy)
時間経過: -2 / 時間
0になると強制睡眠
💛 なかよし度 (bond)
時間経過: -1 / 日
低いと敬語になって距離を取る
全パラメータは 0〜100。時間とともに自然減少し、ユーザーの行動で回復する
