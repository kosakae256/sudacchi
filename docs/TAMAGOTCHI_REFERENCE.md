# たまごっち機能リファレンス

初代（1996年）から最新作（2023年）までの機能を網羅的にまとめた資料。
スダッチ開発の参考用。

---

## 基本アクション（初代で確立）

初代たまごっち（1996年）で確立された8つの基本操作。以降の全シリーズの土台。

| アクション | 内容 | 効果 |
|-----------|------|------|
| **ごはん** | 食事を与える | おなか+1、体重+1 |
| **おやつ** | おやつを与える | ごきげん+1、体重+2 |
| **ゲーム** | 数字当てゲーム（左右どちらが大きいか） | 勝つとごきげん↑、体重-1 |
| **トイレ** | うんちを流す | 放置すると病気の原因 |
| **電気** | 就寝時に消す | 消し忘れ＝お世話ミス |
| **注射（くすり）** | 病気時に投与 | 病気回復 |
| **しつけ** | わがまま時に叱る | しつけメーター+25% |
| **チェック** | ステータス確認 | おなか・ごきげん・しつけ・年齢・体重を表示 |

## ステータスパラメータ

### 初代のパラメータ

| パラメータ | 表示方法 | 増減 |
|-----------|---------|------|
| **おなか** | ハート4個 | 時間で減少。ごはんで+1 |
| **ごきげん** | ハート4個 | 時間で減少。おやつ/ゲームで回復 |
| **しつけ** | 0〜100% | わがまま時のしつけで+25% |
| **体重** | 数値（g） | 食事で増加、ゲームで減少 |
| **年齢** | 日数 | 1日ごとに+1 |

### 後期で追加されたパラメータ

| パラメータ | 登場時期 | 内容 |
|-----------|---------|------|
| **ごっちポイント（通貨）** | 2004〜 | ゲーム等で獲得。ショップで使う |
| **スキル（知力・芸術・社交）** | 2005〜 | 進化分岐・仕事に影響 |
| **友情レベル** | 2004〜 | 通信相手ごとの友好度 |
| **好感度** | 2016〜 | 結婚相手候補との親密度 |

## 成長システム

### 成長段階

| 段階 | 期間目安 | 備考 |
|------|---------|------|
| **たまご期** | 数十秒〜数分 | 操作不可。待つだけ |
| **幼児期（ベビー期）** | 約1時間 | 基本お世話のみ |
| **反抗期（こども期）** | 約1〜3日 | 学校に通う（V4〜） |
| **思春期（ヤング期）** | 約1〜5日 | スキル・パラメータが分岐に影響 |
| **フレンド期（大人）** | 寿命まで | 結婚可能、仕事可能 |

### 進化分岐の条件

進化先のキャラクターは以下の要素で決まる:

- **お世話ミスの回数**: おなか or ごきげんが0の状態で15分放置 → ミス1回
- **しつけ度**: わがまま呼び出しへの対応回数
- **スキルポイント**: 知力・芸術・社交の偏り（V4〜）
- **遊んだゲームの種類・回数**（一部モデル）
- **食事の傾向**（一部モデル）

→ ミスが少ないほど「良いキャラ」に、多いほど「不健康なキャラ」に進化。

## 死亡・病気・家出

### 死亡条件（初代〜2008年頃）

| 条件 | 詳細 |
|------|------|
| **病気放置** | ドクロマーク出現後6時間放置で死亡 |
| **餓死** | おなかゼロで12時間放置 |
| **ストレス死** | ストレスが極限 → 1時間放置でも死亡 |
| **寿命** | 平均15〜25日。お世話の質で変動 |

### 死亡演出
- 墓石・おばけが画面に表示
- リセットで新しいたまごが孵る

### 「家出」への変更（2009年〜）
- たまごっちiD以降、「死亡」ではなく「家出」に変更
- 子供への心理的配慮が理由
- 一部モデルでは寿命制度自体を廃止

### 病気システム
- ドクロマーク表示 → 注射で回復
- 放置すると悪化 → 死亡/家出
- げんきが低い時に発生しやすい

## 通信・ソーシャル機能の進化

```
なし(1996) → 赤外線(2004) → 赤外線+DL(2009) → NFC(2014)
→ Bluetooth(2018) → Wi-Fi(2023)
```

### 通信でできること（世代ごと）

| 世代 | 通信手段 | できること |
|------|---------|-----------|
| 2004〜 | 赤外線 | 遊ぶ・プレゼント交換・対戦ゲーム・結婚 |
| 2009〜 | 赤外線+DL | アイテム/壁紙/ゲームのダウンロード |
| 2014 | NFC | 実店舗タッチスポットでアイテム獲得 |
| 2018〜 | Bluetooth+アプリ | 全国のユーザーとオンラインで交流・結婚 |
| 2023〜 | Wi-Fi | タマバース（仮想空間で世界中と交流） |

## 結婚・繁殖システム（2004年〜）

| 要素 | 内容 |
|------|------|
| **条件** | 友好度/好感度が一定以上 + フレンド期（大人） |
| **結婚** | 通信で相手と結婚 → 赤ちゃん誕生 |
| **世代交代** | 子供を新たに育てる。親は去る |
| **遺伝（2016年〜）** | 両親の外見パーツを子が受け継ぐ（頭・体型・顔・色・尻尾など） |
| **隔世遺伝** | 祖父母・先祖の特徴が出ることも |

遺伝システム（m!x, 2016年）は組み合わせにより無限に近いバリエーションを生む画期的な仕組み。

## ミニゲーム

| 世代 | ゲーム数 | 特徴 |
|------|---------|------|
| 初代 | 1種 | 数字の大小当て |
| V2 | 14種 | ソロ7種 + 通信対戦7種 |
| V4〜 | 多数 | スキル別ゲーム（知力系/芸術系/社交系） |
| カラー世代〜 | 多数 | グラフィック強化。ダウンロード追加も |

ゲームの役割:
- ごきげん回復
- 体重減少
- ごっちポイント獲得
- スキルポイント獲得（V4〜）

---

## アイテム・ショップ（2004年〜）

| 要素 | 内容 |
|------|------|
| **通貨** | ごっちポイント（ゲーム・仕事で獲得） |
| **食べ物** | ごはん系・おやつ系。種類によって回復量が異なる |
| **おもちゃ** | 遊びに使える。ごきげん回復 |
| **アクセサリー** | 着せ替え（iD〜）。見た目カスタマイズ |
| **所持上限** | 最大32個（V2） |
| **入手方法** | ショップ購入 / 通信プレゼント / パスワード / DL |
